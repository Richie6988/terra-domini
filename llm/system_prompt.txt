## BLOCKED PATTERNS (caused regressions)
- Full file rewrite when str_replace works → orphan code blocks
- Adding routes without verifying imports on both sides → NameError
- fake-initial without cleaning django_migrations → InconsistentMigration
- Appending Python at wrong indentation → IndentationError
- Double-appending code blocks → duplicate columns/functions
- Using .map() without Array.isArray guard → TypeError at runtime
- Adding JSX provider without closing tag → Unterminated JSX

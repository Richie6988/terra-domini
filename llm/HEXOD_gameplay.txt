## CORE LOOP
1. Open map → locate free POI territories nearby
2. Claim territory → receive NFT card
3. Territory produces HEX Coin + resources passively (24h cycle)
4. Spend resources → build improvements → increase production
5. Unlock skills → improve attack / defense / economy
6. Attack enemy territories → expand kingdom
7. Daily missions + events keep engagement fresh

## TERRITORY SYSTEM

### H3 Grid
Resolution 10 (~65m hex diameter). 982 POIs seeded from real-world landmarks.
Categories: landmark, religious_site, ancient_ruins, natural_wonder,
            cultural_site, economic_hub, transport_hub, etc.

### Rarity (target distribution)
  common:    60%   standard POIs
  uncommon:  25%   notable landmarks
  rare:      10%   famous worldwide
  epic:       4%   iconic monuments
  legendary:  0.9% world wonders
  mythic:     0.1% unique per region

### Biome System (Köppen-based)
  urban, rural, forest, mountain, coastal, desert, tundra, industrial, landmark
  Assigned at creation from real geography + POI category.
  Each biome produces different primary resources.

### Production Formula (24h cycle)
  output = tdc_per_day × rarity_mult × shiny_bonus × cluster_bonus × building_bonus
  Rarity multipliers: common×1 / uncommon×1.5 / rare×2.5 / epic×4 / legendary×8 / mythic×20
  Shiny bonus: ×1.5
  Cluster bonus: up to ×1.3 (5+ territories same kingdom)

### Claim Rules
  Free: first territory OR adjacent to owned territory
  Paid: tdc_per_day × 10 for non-adjacent
  Cooldown: none in V0

## COMBAT

### 3 Attack Types
1. ASSAULT ⚔️  — brute ATK vs DEF → capture territory
   Cost: res_petrole×20 + res_acier×15
   Formula: win_chance = clamp(0.35 + (atk/def)×0.35, 0.10, 0.90)
   Counter: Fortification + Résistance prolongée skill

2. INFILTRATION 🔓 — Données vs Stabilité → neutralize DEF 6h (no capture)
   Cost: res_donnees×25 + res_composants×10
   Stack: 3+ infiltrations → DEF drops to 15% (vs 50% single)
   Visual: red dashed border on vulnerable hex
   Counter: Tour de contrôle + Cyberguerre skill

3. BLOCKADE 🚫  — Influence vs Influence → -50% production 24h
   Cost: res_influence×30
   Counter: Routes commerciales + Résistance prolongée skill

### Coalition Anti-Dominant
Attacker controls >8% all territories → all defenders get +25% DEF vs that player

### Battle Report
  { title, detail: "ATK X vs DEF Y", loot, tip }

## SKILL TREE

### Fork System (permanent at position 2)
  attack:    assault_mecanique OR cyberguerre_tactique
  defense:   resistance_prolongee OR riposte_automatique
  economy:   specialisation_biome OR empire_diversifie
  influence: soft_power OR propagande_noire
  tech:      militech OR biotech

Rules:
  - Position 0 of each branch: FREE (no cost)
  - Fork is irreversible — other branch locked permanently

## CAMPAIGNS

### Campaign 1: Naissance d'un Empire (J1→J7)
  D1: Claim first territory → unlock Rural biome
  D2: Collect 50 resources → unlock Coastal biome
  D3: Own 3 territories → unlock Assault attack
  D4: Win a battle → free skill attack[0]
  D5: Claim a POI → reveal Mountain biome
  D6: Join/create alliance → unlock Infiltration
  D7: 7 territories in kingdom → title "Fondateur" + Campaign 2

  Harder objectives, unlocks Blockade + advanced skills
